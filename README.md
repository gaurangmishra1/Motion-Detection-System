# Motion Detection Alarm System

## 📌 Overview
This project is a real-time Motion Detection and Object Detection system developed using Python and OpenCV.  
It detects faces, eyes, motion, and blue-colored objects using a webcam and triggers an alarm sound when motion or specific objects are detected.

---

## 🚀 Features

- 👤 Face Detection using Haar Cascade
- 👁️ Eye Detection
- 🎯 Motion Detection using Background Subtraction
- 🔵 Blue Color Object Detection
- 🔔 Alarm Sound Trigger System
- 📷 Real-time Webcam Monitoring

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy
- Pygame
- Haar Cascade Classifiers

---

## How to Run the Project

Step 1 — Install Dependencies

pip install -r requirements.txt  

Step 2 — Run the Program

python mdc5.py  

Step 3 — Stop Program

Press **q** to exit the camera window.

## How It Works

1. Webcam captures live video.  
2. Faces and eyes are detected using Haar Cascade classifiers.  
3. Motion detection is performed using background subtraction.  
4. Blue-colored objects are detected using HSV color detection.  
5. When motion or object detection occurs, an alarm sound is triggered.
